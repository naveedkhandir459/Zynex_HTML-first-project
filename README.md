HTML Course Website
Project Overview

This is a simple HTML-only website created as part of a web development course.
The project focuses on practicing basic HTML concepts by building a small website without using CSS or JavaScript.

Features

Home section with site introduction and 3 sample blog posts

About section with details about the website and a performance table

Contact section with a simple contact form (name, email, message)

Technologies Used

HTML5 only

How to Run

Open the Home.html file in any web browser.
